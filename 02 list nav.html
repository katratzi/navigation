Code Notes

In this case we turn the navigation into an unordered list <ul>, this is a good semantic structure for what is in effect a list of links. In order to remove the normal ● circle we use some css to, in this case, remove all styling. And additionally we add in our own little decorator with a css 'content' setting.

// html unordered list <ul> <li><a href="#">Home</a></li> <li><a href="#">Information</a></li> <li><a href="#">Contact Us</a></li> <li><a href="#">FAQ</a></li> <li><a href="#">Company Factsheet</a></li> </ul> /* set list style and also center for neatness */ ul { list-style-type: none; text-align: center; } /* display inline makes them appear on the same line */ li { display: inline; } /* add my own decorator for nav, but not for the 1st child in list */ li:before { content: " ~ "; } li:first-child:before { content: ""; }
